# 归田园居

## 项目简介
《归田园居》是一款基于 Vue 3 + Vite 的治愈系田园生活模拟游戏。玩家辞去大城市的工作回到家乡，通过耕种、经营市集、装饰房屋、烹饪美食以及与儿时玩伴（NPC）互动，体验温馨的乡村日常。

## 核心玩法
- 🌱 **田园种植**：在农场种植各类作物，收获后存入仓库
- 🛒 **市集交易**：关注每日行情波动，择时出售作物获取收益
- 🍳 **烹饪系统**：使用收获的食材制作美味菜肴
- 🎁 **NPC社交**：向4位儿时玩伴赠送料理，提升好感度
- 🏠 **小屋装修**：购买家具装饰自己的温馨小窝
- 📖 **剧情解锁**：随着等级提升，解锁与伙伴们的回忆故事
- 🎲 **每日Buff**：肉鸽玩法，每天三选一的随机加成效果

## 技术栈
- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **状态管理**：Pinia
- **数据持久化**：LocalStorage

## 快速开始
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构
```
src/
├── components/      # 通用组件
├── stores/          # Pinia 状态管理
│   ├── game.js      # 游戏核心逻辑
│   └── npc.js       # NPC 数据与好感度
├── views/           # 页面视图
├── App.vue          # 根组件
└── main.js          # 入口文件
```

## 注意事项
1. **美术资源规范**：所有新增美术资源必须存放于 `public/assets/`，且必须经过透明化处理
2. **响应式布局**：游戏界面适配 PC（9:16 居中）和移动端（全屏）
3. **状态管理**：所有游戏核心数据通过 Pinia Store 管理
4. **数据持久化**：关键数据变更时自动写入 LocalStorage
