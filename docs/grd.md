# 游戏需求文档 (GRD) - 归田园居

## 1. 项目概况

| 项目 | 内容 |
| :--- | :--- |
| **项目名称** | 归田园居 |
| **游戏类型** | 模拟经营 / 角色扮演 / 剧情养成 |
| **美术风格** | 治愈系日式手绘风（清淡线条，赛璐璐平涂）|
| **核心体验** | 返乡种田，振兴家乡，与儿时玩伴重逢，体验温馨的乡村日常 |

## 2. 产品定位与核心循环

### 2.1 游戏定位
本作是一款**返乡题材**的田园生活模拟游戏。

**背景故事**：
在大城市打拼多年的你，因为厌倦了996的高压生活，决定辞职回到阔别已久的故乡。虽然这里没有城市的灯红酒绿，但有清新的空气、肥沃的土地，还有儿时一起长大的伙伴们。

### 2.2 核心玩法循环
1. **劳作（Input）**：耕种荒地 → 种植蔬菜 → 细心浇水 → 收获农产品
2. **经营（Process）**：观察市场行情 → 择时出售作物 → 获得金币与经验
3. **建设（Output）**：购买家具装饰老屋 / 提升等级解锁更高级的作物
4. **情感（Reward）**：烹饪美食赠送NPC → 提升好感度 → 触发连贯的事件剧情

## 3. 核心系统详解

### 3.1 田园种植系统 (Farming)

#### 作物列表
| 作物 | 种子价格 | 生长时间 | 售价（基准）| 解锁等级 |
| :--- | :---: | :---: | :---: | :---: |
| 白萝卜 | 10 | 10秒 | 20 | Lv.1 |
| 金土豆 | 15 | 20秒 | 35 | Lv.1 |
| 红番茄 | 20 | 30秒 | 50 | Lv.2 |
| 甜玉米 | 25 | 40秒 | 65 | Lv.5 |
| 大南瓜 | 40 | 60秒 | 100 | Lv.7 |
| 神秘种子 | 10 | 25秒 | 随机 | Lv.1 |

#### 神秘种子产出
- 60% 概率：杂草（固定售价 1 金币）
- 30% 概率：随机普通作物
- 10% 概率：金苹果（固定售价 500 金币）

#### 金牌作物
- 收获时有 20% 概率获得金牌作物
- 金牌作物经验值 ×3
- 带有特殊金色光效

### 3.2 周期与Buff系统

#### 周期机制
- 每天可耕种 8 次
- 手动点击"下一天"推进
- 每天开始时选择一个Buff

#### 每日Buff（三选一）
| Buff名称 | 效果 |
| :--- | :--- |
| 太阳当空照 | 作物生长速度 +20% |
| 小道消息 | 红番茄和甜玉米收购价 +20% |
| 临时扩容 | 仓库容量 +10 格 |
| 邻里互助 | 种子价格 -20% |
| 肥沃土壤 | 收获经验值 +30% |
| 通货膨胀 | 出售价 +50%，种子价翻倍 |
| 疯狂生长 | 生长速度 +200%，20%枯萎变杂草 |
| 黑市交易 | 金土豆 ×3，其他作物 1 金币 |
| 透支体力 | 收获翻倍，无经验 |
| 投机者 | 神秘种子金苹果概率提升至20% |
| 金手指 | 金牌作物概率 50% |
| 变废为宝 | 杂草售价 50 金币 |
| 期货合约 | 立即 +200 金币，明日价格减半 |
| 精挑细选 | 仓库 -10 格，售价 +30% |
| 神秘客的馈赠 | 每卖10个作物送1颗神秘种子 |

### 3.3 市集系统 (Market)

#### 价格波动
- 每日价格在基准价的 50%-200% 之间波动
- 红色/绿色标识涨跌
- 7日行情走势图可视化

#### 仓库系统
- 容量：30 格（受Buff影响）
- 仅存储作物，种子不占位

### 3.4 烹饪系统 (Cooking)

#### 菜谱
| 食材组合 | 菜肴 | NPC最爱 |
| :--- | :--- | :--- |
| 萝卜 + 番茄 | 茄汁萝卜 | 小白 |
| 土豆 + 番茄 | 酸甜土豆丝 | 小绿 |
| 番茄 + 玉米 | 红油玉米 | - |
| 番茄 + 南瓜 | 甜甜南瓜泥 | - |

#### 烹饪品质
- 通过进度条小游戏决定
- 两次命中：美味（+10好感）
- 一次命中：普通（+5好感）
- 零次命中：灾难（-5好感）
- 赠送NPC最爱菜肴好感度翻倍
